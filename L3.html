Built in Data Types
Variables can store data of different types, and different types can do different things.
for example Text type variable, string. Can store words like "cat" or "dog" and int or float cannot.
Here are some of them:

Text Type: str
Numeric Types: int, float
Sequence Types: list, tuple, range
Mapping Type: dict

Example Data Type
x = "Hello World" string
x = 15 int
x = 15.5 float
x = ["orange", "banana", "apple", "Tomato"] list
x = ("orange", "banana", "Tomato") tuple
x = range(6) range
x = {"name" : "Alex", "age" : 29} dictionary

Print() - is responsible for printing a given element in the console
Example:

print ( " Hello World! " )

console output:


Hello World!

input() - is responsible for reciving user-entered data
Example:

name = input ( " What's your name? " )
print ( " My name is " , name )

Let's assume that the name entered is "John"
console output:

My name is John
